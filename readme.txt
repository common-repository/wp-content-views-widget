== Description ==

= Query and Display Wordpress Content or Category Terms in a Widget for your Wordpress site with Wordpress Content Views Widget =
<p>Simple widget that queries the database and allows you to display content in any page or sidebar.</p> 
<p>No coding required - Mobile Friendly.</p> 
<p>Our Purpose is to enhance the way you can dynamically query & display content, like Views Module is doing for Drupal platform, but in WordPress terms.</p> 

**SUPPORTS**
* Content Types: Posts, Pages
* Taxonomies: Post Categories, Post Tags
* Fields: Title, Content, Excerpt, Featured Image
* Order: Descending of Ascending Order
* Order By: id, author, title, name, date, random, none
* Number of Posts to display
* Styling: List View, Select number of columns per row, Content Alignment


**PRO VERSION**

* Choose Any WordPress Custom Post Type to query & display
* Choose Any WordPress Custom Taxonomy to display
* Ability to Choose which Specific Post to display
* Choose Slideshow View to display the Content
* Choose Animations
* Woocommerce Compatible


[Get it Here](https://webdeveloping.gr/product/wordpress-content-views-widget-pro "Query & Display with Wordpress Content Views Widget Pro")

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go and use it in the widget screen
4. Add a title, choose what to display, style it, assign it to a region and it work for you out of the box!
